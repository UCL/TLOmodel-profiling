This is a repository for storing profiling outputs from the TLOmodel, an epidemiology modelling framework for the Thanzi la Onse project.

Running Simulations with the Model¶

For more information about setting up, using, and running simulations using the model, please see the main repository and information provided there.

View the Profiling Results¶

Profiling results are hosted on GitHub pages and are built using sphinx and some custom Python scripts to process the results data.
